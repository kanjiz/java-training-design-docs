# TwoFruitsPrinter 詳細設計書

## 1. 機能要件

- 2つの文字列を別々の行に表示するプログラム
- 1行目に「モモ」を表示
- 2行目に「ミカン」を表示

### 3.1 mainメソッド

| 項目 | 内容 |
|------|------|
| 修飾子 | public static |
| 戻り値の型 | void |
| メソッド名 | main |

#### 3.1.2 機能説明

- プログラムのエントリーポイント
- 1行目に「モモ」を表示

- 各文字列の出力後に改行を行う

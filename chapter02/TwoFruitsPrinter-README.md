# TwoFruitsPrinter 詳細設計書

## 1. 機能要件

- プログラム内に直接記述した2つの果物名を画面に表示する
- キーボード等からの入力は行わない
- 1行目に「モモ」、2行目に「ミカン」を表示する

## 2. クラス設計

### 2.1 クラス図

|------|------|
| クラス名 | TwoFruitsPrinter |
| パッケージ/名前空間 | なし（デフォルトパッケージ） |
| 修飾子 | public |

## 3. クラス図の各操作が実現すべき処理

### 3.1 main

- 1行目に「モモ」を表示
- 2行目に「ミカン」を表示
- 各文字列の出力後に改行を行う
